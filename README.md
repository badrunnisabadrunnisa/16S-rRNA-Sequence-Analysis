# 16S rRNA Sequence Analysis of Bacterial Species

## Overview

This project demonstrates the use of bioinformatics tools to analyze bacterial 16S rRNA sequences. Sequence similarity, conserved regions, and evolutionary relationships were investigated using publicly available sequence data.

## Objectives

* Retrieve 16S rRNA sequences from NCBI GenBank
* Perform sequence similarity analysis using BLAST
* Conduct multiple sequence alignment using Clustal Omega and T-Coffee
* Identify conserved regions among bacterial species
* Analyze phylogenetic relationships

## Bacterial Species

* Escherichia coli
* Salmonella enterica
* Shigella flexneri
* Klebsiella pneumoniae
* Enterobacter cloacae

## Tools Used

* NCBI GenBank
* NCBI BLAST
* Clustal Omega
* T-Coffee

## Methodology

1. Downloaded bacterial 16S rRNA sequences from NCBI.
2. Performed BLAST analysis to identify similar sequences.
3. Conducted multiple sequence alignment using Clustal Omega.
4. Verified alignment using T-Coffee.
5. Examined conserved regions.
6. Generated phylogenetic relationships.

## Results

* High sequence similarity was observed among the selected bacterial species.
* Conserved nucleotide regions were identified across aligned sequences.
* Phylogenetic analysis demonstrated evolutionary relationships among the bacterial species.

## Conclusion

The project demonstrates how 16S rRNA sequence analysis can be used for bacterial identification and evolutionary studies using bioinformatics tools.
